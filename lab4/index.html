¿Qué diferencias y semejanzas hay entre Java y JavaScript?
Java es un programa del paradigma de objetos, al igual que JavaScript. JavaScript, a diferencia de Java es un lenguaje de alto nivel y es de "scripting" dado que puedes correr cualquier comando sin compilar.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

 Date(); 

 //Obtiene la fecha completa en ISO 8601
 Date.now();
